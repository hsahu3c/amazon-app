# Shopify-Amazon plan module - documentation

This documenation is the overall explanation of the plan module used in Amazon Multichannel.

> Features supported:

```json

    1. Monthly and yearly plan subscription
    2. Plans based on category
    3. Plan restrictions based on order_sync usage credits
    4. Recommendations based on app usage, monthly usage and historical data recommendations.
    5. Excess usage charges
    6. Mails
    7. Mail settings for plan
    8. Custom plans
    9. Custom Payments
    10. Plan renew
    11. Plan deactivation
    12. Webhooks
    13. Transaction history
    14. Dynamo syncing enable and disable
    15. Plan restoration
    16. AddOn support

```
